# TLSNotary Templates

This directory stores seeded template JSON files.

- All templates are currently in `draft` status.
- Schema: `../schema/template.schema.json`
- Index: `./index.json`

## Conventions

- Keep claims narrow and verifiable.
- Minimize disclosure output.
- Redact all account identifiers and secret-bearing fields.
- Include freshness policy to avoid stale proofs.
